PROJECT TITLE:Project 1: Web Server/Client
PURPOSE OF PROJECT: Simulate a simplified web server/client communication protocol
VERSION or DATE: 2/20/17
HOW TO START THIS PROJECT: Open two terminals, and navigate to the correct directory.
In one terminal, run java ServerApp "propogation delay" "transmission delay"
Add the prop and trans delays that you want. If no values are included, will default to zero
In the other terminal, run java ClientApp "version"
Add the version that you want. If not included, or if value is not 1.0 or 1.1, we default to 1.1
Type any requested file names. Files need to be .tml files.
AUTHORS:Jack Plumb and Austin Wiles
USER INSTRUCTIONS:
To turn caching on or off, go into Client.java and change the boolean at the top
To turn auto-embedding on or off, go into Server.java and change the boolean at the top
